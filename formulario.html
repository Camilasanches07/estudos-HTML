Formulário Aqui

tag "input não tem fechamento (/input)"

Utilizamos " placeholder" para dizer ao usuário o que fazer. Ex:Digite seu nome aqui

Utilizamos " label" para dizer ao usuário o que é aquele campo. Ex: Nome

Utilizamos dentro do input o "id=nome do campo" para especificar que a legenda "Nome" está relacionada ao campo de input do tipo texto

Utilizamos o type "password" para digitarmos senha, assim, quando digitarmos sairá com o campo escondido

Utilizamos o type "email" para solicitarmos o email do usuário

Utilizamos o type "submit" para enviarmos o Formulário

Utilizamos "required" para que a pessoa preencha todos os campos, sendo impossível enviar o formulário sem ter preenchido

Utilizamos o type "radio" para ter como marcar somente uma opção

Utilizamos o type "checkbox" para ter como marcar mais de uma opção

Utilizamos o type "select" para dar múltiplas escolhas ao usuário

Utilizamos "textarea" para o usuário escrever um texto, e "rows" para definir quantas letras poderá ser escritas, "rows" para número de colunas
