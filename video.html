Página de Vídeo

O nome da tag é "video", dentro dela também haveráum "source src" para mostrar onde está o vídeo

Se quiser deixar uma imagem no início do vídeo, usar tag "poster"

Utiliza-se o "autoplay" para quando entrar na página o vídeo começar a rodar automaticamente

Utiliza-se "controlslist="nodownload" para imposibilitar o usuário de fazer dowload

Utilizase "disablepictureinpicture" para desabilitar os 3 pontinhos

Utiliza-se "nofullscreen" para desabilitar tela cheia
