Links

Utilizamos a tag "a href="link do site" Ex: Ir para o Google

Utilizamos a tag "a href="link do site"target="_blank"" para abrir o link em uma nova guia Ex:Ir para o Google

Para ir em alguma página alternativa do site, basta indicar o nome do arquivo da página "a href="sobre.html" exemplo: Sobre Nós

Para navegar em um arquivo de uma pasta, exemplo que criei "equipes", basta digitar "a href="Nome da pasta/nome do arquivo. Ex:Camila

Para voltar da página "Camila" para essa principal, deve escrever "a href="../nome do arquivo de deseja"

Para voltar 2 pastas para trás , use a estrutura "a href="../../nome do arquivo de deseja"

Imagens

Para adicionar imagens utilize a tag "img src="", src=caminho da imagem, não precisa de fechamento de tag, para diminuir o tamanho uitilize "width" e "height". "alt" é utilizado para descrever a imagem

Para clicar nessa foto dos cachorros e ser direcionado para alguma página específica, basta mandar o link igual o exemplo a baixo

Para adicionar uma imagem que esteja salvo no seu computador, "img src="nome da pasta / nome da foto

Para reduzir o tamanho das imagens e deixar o site mais e=leve, entre em sites como tinyipg, easy resize, para redimensionar as imagens e deixae mais leve


Para adicionar imagens "png" , faça o mesmo caminho. Site photopea é para criar imagens png


Para adiconar gifs basta seguir o mesmo caminho das outras imagens
